# Unit Converter

**Category:** Calculators
**URL:** https://delphi.tools/tools/unit-converter
**Status:** stable

## What It Does

Converts values between units across multiple categories: length, weight, temperature, and data storage.

## When to Use

- You need to convert a measurement between metric and imperial units (e.g. kilometres to miles).
- You are working with data sizes and need to convert between bytes, kilobytes, megabytes, gigabytes, etc.
- You need a temperature conversion (Celsius, Fahrenheit, Kelvin).

## Browser Mode (Default)

### Inputs

- **Value input field** — enter the numeric quantity to convert.
- **Category selector** — choose the measurement category (length, weight, temperature, data).
- **From-unit selector** — choose the unit of the entered value.
- **To-unit selector** — choose the target unit for conversion.

### Step-by-Step

1. Navigate to https://delphi.tools/tools/unit-converter
2. Click the category selector and choose the measurement type (e.g. "Length").
3. Click the from-unit selector and choose the unit you have (e.g. "Kilometres").
4. Click the to-unit selector and choose the unit you need (e.g. "Miles").
5. Click the value input field and type the numeric quantity.
6. The converted result appears instantly in the output area below the selectors.

### Output

The converted value displayed below the input controls, labelled with the target unit.

### Options

- **Category selector** — switches the available units in the from-unit and to-unit selectors.
- **From-unit and to-unit selectors** — independently configurable; swap them manually to reverse the conversion direction.
