# Palette Collection

**Category:** Colour
**URL:** https://delphi.tools/tools/palette-collection
**Status:** stable

## What It Does

Displays a curated library of colour palettes organised into 10 themed categories. Click any palette to expand it and see individual hex values, which can be copied with a single click.

## When to Use

- You need inspiration for a colour palette for a design project.
- You want to find a palette that matches a specific mood, era, or style.
- You need ready-made hex values from a named palette without generating one yourself.

## Browser Mode

### Inputs

- **Category tabs** — 10 categories: Classic, Nature, Keycaps, Vintage, Modern, Bold, Soft, Monochrome, Seasonal, Artistic.
- No text input or file upload is required.

### Step-by-Step

1. Navigate to https://delphi.tools/tools/palette-collection
2. Click a category tab to browse palettes in that category.
3. Scroll through the palette grid. Each palette card shows a colour strip preview and a name.
4. Click a palette card to expand it and view all individual colours with their hex values.
5. Click any hex value to copy it to the clipboard.

### Output

- **Palette cards** — colour strip previews with palette names displayed in the grid.
- **Expanded palette view** — individual colour swatches with their hex values shown when a card is clicked.
- **Clipboard copy** — clicking a hex value copies it instantly.

### Options

- Category tabs filter the displayed palettes to one of 10 themes: Classic, Nature, Keycaps, Vintage, Modern, Bold, Soft, Monochrome, Seasonal, Artistic.

## CLI Mode (Node.js)

N/A — browse-only tool with no programmatic API. Use Browser Mode.
