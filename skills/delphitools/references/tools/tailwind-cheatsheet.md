# Tailwind Cheatsheet

**Category:** Other Tools
**URL:** https://delphi.tools/tools/tailwind-cheatsheet
**Status:** stable

## What It Does

Provides a static quick-reference for Tailwind CSS utility class names, organised by category. Search by class name or CSS property and click any entry to copy it.

## When to Use

- You cannot remember the exact Tailwind class name for a CSS property.
- You want to browse all available classes in a category (e.g. all spacing utilities).
- You need to quickly copy a class name without opening the full Tailwind docs.

## Browser Mode (Default)

### Inputs

- **Search field** — type a Tailwind class name (e.g. "flex-row") or a CSS property (e.g. "justify-content") to filter the reference table.
- **Category sidebar** — click a category name to jump to that section.

### Step-by-Step

1. Navigate to https://delphi.tools/tools/tailwind-cheatsheet
2. To find a specific class, type in the search field. Results filter instantly to matching class names and CSS properties.
3. To browse by topic, click a category in the sidebar: Layout, Flexbox, Grid, Spacing, Sizing, Typography, Backgrounds, Borders, Effects, or Transitions.
4. Scroll through the matching entries. Each row shows the Tailwind class name and the CSS property/value it applies.
5. Click a class name to copy it to the clipboard.

### Output

- **Filtered reference table** — rows showing Tailwind class name and corresponding CSS output.
- **Clipboard copy** — clicking a class name copies it instantly.

### Options

- Category sidebar provides quick navigation to one of 10 sections: Layout, Flexbox, Grid, Spacing, Sizing, Typography, Backgrounds, Borders, Effects, Transitions.

## Advanced Mode (Node.js/CLI)

N/A — static reference page with no programmatic interface. Use Browser Mode.
