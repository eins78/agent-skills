# Typographic Unit Calculator

**Category:** Typography and Text
**URL:** https://delphi.tools/tools/typo-calc
**Status:** stable

## What It Does

Converts a value between traditional typographic units — agates, ciceros, picas, points (pt), inches, and millimetres — displaying the result in all other units simultaneously.

## When to Use

- You are working with print or publishing specifications that use picas or points and need metric equivalents.
- You have a measurement in millimetres or inches and need the typographic unit equivalents.
- You need to reconcile dimensions across screen (pt/px) and print (mm/inches/picas) workflows.

## Browser Mode (Default)

### Inputs

- **Value input field** — enter the numeric quantity to convert.
- **Source unit selector** — choose the unit of the entered value (agates, ciceros, picas, pt, inches, mm).

### Step-by-Step

1. Navigate to https://delphi.tools/tools/typo-calc
2. Click the value input field and type the numeric value you want to convert.
3. Click the source unit selector and choose the unit that matches your input.
4. The results panel updates instantly, showing the equivalent value in all other supported units.
5. Read off the converted values from the results panel as needed.

### Output

A results panel with one row per unit (agates, ciceros, picas, pt, inches, mm), each showing the converted value for the entered quantity.

### Options

- Source unit selector — change the input unit at any time; the results panel recalculates immediately.
