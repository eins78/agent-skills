# Harmony Genny

**Category:** Colour
**URL:** https://delphi.tools/tools/harmony-genny
**Status:** stable

## What It Does

Generates colour harmony sets from a base colour, covering relationship types such as complementary, triadic, split-complementary, analogous, tetradic, and square.

## When to Use

- You need a harmonious multi-colour palette built around a specific brand colour.
- You are exploring which colour relationships (complementary vs. triadic, etc.) best suit a design.
- You need the hex codes for all colours in a harmony set to use in a design tool or code.

## Browser Mode (Default)

### Inputs

- **Base colour hex input field** — enter the starting colour as a hex value (e.g. `#e63946`).
- **Harmony type selector** — choose the harmony type (complementary, triadic, split-complementary, analogous, tetradic, square, etc.).

### Step-by-Step

1. Navigate to https://delphi.tools/tools/harmony-genny
2. Click the base colour hex input field and type or paste your base colour hex value.
3. Click the harmony type selector and choose the desired harmony relationship.
4. The harmony colour set updates instantly, showing all colours in the selected harmony.
5. Click the Copy button next to any colour swatch to copy that colour's hex value to the clipboard.

### Output

A set of colour swatches — the number varies by harmony type (2 for complementary, 3 for triadic, etc.) — each labelled with its hex value and accompanied by a Copy button.

### Options

- Harmony type selector controls which colour relationship is used to derive the output colours.
