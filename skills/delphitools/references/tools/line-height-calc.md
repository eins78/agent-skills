# Line Height Calculator

**Category:** Typography and Text
**URL:** https://delphi.tools/tools/line-height-calc
**Status:** stable

## What It Does

Calculates the recommended line height for a given font size, applying typographic best-practice ratios to produce comfortable reading rhythm.

## When to Use

- You have a font size and need to determine an appropriate CSS `line-height` value.
- You are setting up a type scale and want consistent, readable line spacing across all sizes.
- You want to verify that a proposed line height falls within accepted typographic guidelines.

## Browser Mode (Default)

### Inputs

- **Font size input field** — enter the font size in pixels (e.g. `16`).

### Step-by-Step

1. Navigate to https://delphi.tools/tools/line-height-calc
2. Click the font size input field and type the font size in pixels.
3. The recommended line height value appears instantly in the output area.
4. Use the displayed line height value directly in your CSS `line-height` property.

### Output

The recommended line height value (typically a unitless ratio or pixel value), displayed below the font size input.

### Options

None — the tool applies standard typographic ratios automatically based on the entered font size.

## Advanced Mode (Node.js/CLI)

N/A — custom implementation, use Browser Mode.
